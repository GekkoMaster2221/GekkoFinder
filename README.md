================================================================================
                              GEKKOFINDER v2
                   WiFi Network Security Assessment Tool
================================================================================

GekkoFinder is built for real-time WiFi security evaluation. It helps users assess encryption protocols, signal strength, DNS setup, and unsafe networks (e.g., CUE or traps). Designed for security professionals, it offers accurate safety scores and tools for informed decisions. 

Main Features
- Live WiFi scanning and safety analysis with color-coded scores.
- Commands for network diagnostics, penetration testing, and more.
- Honeypot (unsafe/CUE) detection ensures safe connectivity.
- Compatible with UNIX, Linux-based systems.

INSTALLATION TIPS
   Install dependencies to support bash-like terminal functionality.
